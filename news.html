Sprint 1
During the 1st Sprint we had to:
1. Brainstorm design elements for our project that will be used to create a platform which will help Sememester 3 students. We needed to create a group identity such as logo and name, bring 3 Concepts to the table, Documentation, ...)
2. Make a class diagram for the interaction of all involved stakeholders.
3. Reasearch for the base of our concept.
4. Prepare a video presentation to present our 3 concepts.
Sprint 2

1.Research and re-define the concept. We used different research methods in order to find the best option for students
2. We created a blog where we are going to update the users and lectures about features and sprints
3. Created a storyline from survey and interview. Refined the goal of the website, used personas POV & HMW. We build a business plan which was created with feedback from user studies.
4. The biggest feature for this sprint is the prototype. We invested a lot of time in the prototype. Big improvements were made from the wireframes and sketches.
Sprint 3
During the 3th Sprint we:

1.Created A MoSCoW analysis, in which we created our main purposes for the website
2. We updated our website, creating the login adn forum function
3. We hosted our website on hera


Sprint 4
During the 4th Sprint we had to:
1. Finish up the website
2. We did some user task scenarios with 3 students
3. We wrapped up the blog with all of the updates
4. Prepare a video presentation to present our project.
5.We created the final version of the Concept Document, Interaction Design and a Test Plan
